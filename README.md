### 💰 Expense Tracker Console Application

A console-based expense tracking system built using Java and Collections Framework concepts.
This project helps users manage and track their daily expenses directly through the terminal.

---

## ✨ Features

* ➕ Add Expenses
* 📋 View Expense Records
* 🗑 Delete Expenses
* 🔍 Search Expenses
* 📂 Category-based Expense Management
* 📊 Expense Summary Tracking
* 💾 Data Handling using Java Collections
* 🖥 Simple Console-Based Interface

---

## 🛠 Technologies Used

* ☕ Java
* 📚 Java Collections Framework
* 🧠 Object-Oriented Programming (OOP)

## 🚀 How to Run

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Hacker-Hima/personal-expense-tracking-system-console.git
```

---

### 2️⃣ Open Project

Open the project using:

* VS Code
* IntelliJ IDEA
* Eclipse

---

### 3️⃣ Compile Java Files

```bash
javac *.java
```

---

### 4️⃣ Run Application

```bash
java Main
```
